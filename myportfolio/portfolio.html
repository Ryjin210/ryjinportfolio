College Thesis, Web Systems

THESERVE: ONLINE ORDERING SYSTEM FOR AMARAH'S CORNER - LAS PIÑAS

Through my college thesis project, We've developed THESERVE, an advanced online ordering system tailored for Amarah's Corner in Las Piñas. My role involved meticulously managing the back-end infrastructure and integrating Lalamove Delivery API seamlessly into the platform.

THESERVE redefines the dining experience, offering patrons a user-friendly interface to browse menus, place orders, and coordinate pickups or deliveries with ease.

Front-End Development

Features a User-Friendly Interface, Minimalistic UI

The front-end of the system was skillfully developed using HTML and CSS, enabling the creation of a polished and visually engaging user interface. JavaScript, along with jQuery and AJAX libraries, was harnessed to implement dynamic functionalities, providing users with seamless interactions and real-time updates. Bootstrap elements were thoughtfully integrated to ensure responsiveness and to enhance the system's modern and minimalist aesthetic, contributing to its overall appeal.

Back-End Development

Streamlined Database, Back End API Integration

The back-end was developed using PHP for server-side scripting, handling logic, form data processing, and generating dynamic content. MySQL managed the database contents, enabling seamless interaction with PHP scripts for efficient data management and retrieval. Back-end also features API integration of the delivery orders, which is integrating the Lalamove Delivery API into the system's back-end core.
